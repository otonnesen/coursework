# MATH 423 - Graph Theory

An introduction to the combinatorial, algorithmic, and algebraic aspects of graph theory.

## Topics
- Trees and distance
- Matchings and factors
- Connectivity and paths
- Colouring of graphs
- Planar graphs
- Edges and cycles

## Special topics
- Minimum-weight spanning trees
- Stable matchings
- Line graphs
- Interval graphs
- Split graphs
